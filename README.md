# Forge Assignments

Forge Assignments is an online platform where students from all over MANIT can share their assignments and get them completed on time. Students can find assignments on our platform, complete it and get reward in form of money.

# Development
## Frontend
The front end of the web application has been designed using HTML, CSS, Vanilla Javascript and framework like Tailwind CSS.
It consists of:
Home Page: Introductory page of the website
Find Page: Webpage to find assignments hosted by other students.
Host Page: Host your own assignment
Contact Page: A standard contact us page
View Assignment Page: View Information about a particular assignment



## Backend
The back end of the web application has been developed using PHP and MySQL Database. Using POST and GET methods of request, SQL query is executed. 
While submitting, POST requests are executed and the queriezs are executed according to the input given by the user

## Features

- Responsive Website
- Smooth Assignment Hosting
- Connect with Students all over Campus
